# Cactus

Strategies for farming cactus using various sorting algorithms.

## [Single Drone](./Single%20Drone/)
- [Shear Sort](./Single%20Drone/shear_sort.py): Alternating row and column sorts for grid arrangement
- [Gradient Bubble Sort](./Single%20Drone/gradient_bubble_sort.py): Enhanced bubble sort with incremental adjustments
- [Heapify](./Single%20Drone/heapify.py): Heap-based sorting approach for cactus arrangement
- [Insertion Sort](./Single%20Drone/insertion_sort.py): Classic insertion sort implementation
- [Naive 4-way Bubble Sort](./Single%20Drone/naive_4_way_bubble_sort.py): Multi-directional bubble sort
- [Naive Bubble Sort](./Single%20Drone/naive_bubble_sort.py): Standard bubble sort algorithm
- [Naive Cocktail Sort](./Single%20Drone/naive_cocktail_sort.py): Bidirectional bubble sort variant
- [Oneshot](./Single%20Drone/oneshot.py): Single execution sorting strategy
- [Planting](./Single%20Drone/planting/): Planting helper functions

## [Multi Drone](./Multi%20Drone/)
Currently no strategies available.

Refer to each subfolder for detailed information and usage examples.
